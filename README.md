# my_connect_4  
Basic connect 4 in C in two versions, made during my first year studying C  

## How to use?  
> Help menu :  
> ./my_connect_4 -h  

> To start in **CSFML** :  
> ./my_connect_4 -c  

> To start in **ncurses** :  
> ./my_connect_4 -n
